# SkuFlow Marketing Site Spec

## Brand Positioning

- One-line value proposition: SkuFlow helps multi-location retailers forecast demand and optimize replenishment so they increase availability with less excess inventory.
- Subheadline: Built for planning, merchandising, and operations teams that need faster, more accurate inventory decisions across stores, channels, and suppliers.
- Differentiated pillars:
  - Forecasts tied to execution
  - Exception-first operations
  - Cross-network visibility
  - Scenario confidence
  - Role-specific accountability
- Tone: Confident, modern, operational, ROI-focused.

## Design system

- Typography scale:
  - H1: 48/56
  - H2: 36/44
  - H3: 24/32
  - Body: 16/26
  - Caption: 12/18
- Spacing: 4px base scale, with 16/24/32/48/64 section rhythm.
- Buttons:
  - Primary: Filled blue, white text
  - Secondary: Dark bordered
  - Min touch target: 44px
- Cards:
  - Radius 12–16px
  - Elevated dark-neutral surfaces
  - Heading + concise body + next action

- "Move from disconnected planning files to connected execution..."
- "Replace reactive issue discovery with prioritized exception workflows..."
