# SkuFlow Backend

Custom Product Commerce Platform - Backend Server

## Setup

```bash
npm install
```

## Environment Variables

```bash
cp .env.example .env
```

## Database

```bash
# Run migrations
npm run db:migrate

# Seed sample data
npm run db:seed

# View database
npm run db:studio
```

## Development

```bash
npm run dev
```

## Production

```bash
npm run build
npm start
```
